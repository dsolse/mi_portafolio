Ideación de idea de negocios con design thinking

Ideación de Chubby, un servicio para restaurantes locales emprendedores enfocado en una app a fin de publicar sus platillos y usar algoritmos para mostrar sus platillos a clientes potenciales acorde a sus gustos. Se usó la metodologíá design-thinking siguiendo sus cuatro pasos para crear una idea de negocio a partir de una necesidad: empatizar con usuarios disgustados por la monotonía de sus opciones de comida y su poco tiempo para buscar nuevas opciones acorde a sus preferencias; después se procedió a definir el problema estudiando más a fondo a los usuarios en busca de sus necesidades; a continuación, se idea la solución viendo las mejores alternativas, y finalmente, hacemos un prototipo que testeamos con el usario de manera iterativa hasta satisfacer sus expectativas y solucionar su problema.

Codificador de mensajes con python

Creación de una aplicación de consola de python para cofidifcar y decodificar mensajes. El proyecto se baso programación orientada a objectos, se creó una clase que contiene dos métodos, uno para encriptar y otro para desencriptar otro mnesaje; el funcionamiento de la aplicaicón era reemplazar cada character con otro que era equivalente al anterior pero desplazado cualquier cantidad de posiciones. Asimismo, se automatizó el programa a fin de agregar los archivos a trabajar en una carpeta, usando las funciones de python para manipular directorios del sistema operativo.
